Вход через Telegram

Вариант 1: Вход через Telegram бота

Этот метод требует взаимодействия с нашим Telegram ботом:

  1. Нажмите на кнопку ниже, чтобы открыть нашего Telegram бота.
  2. Если вы используете бота впервые, возможно, потребуется нажать 'Старт'.
  3. Бот предоставит вам кнопку для входа.
  4. Нажмите на эту кнопку, чтобы завершить процесс входа.

Вариант 2: Вход через виджет Telegram

Этот метод использует виджет Telegram для быстрого входа:

  1. Нажмите на кнопку входа через Telegram ниже.
  2. Откроется виджет Telegram, запрашивающий ваш номер телефона.
  3. Введите номер телефона, привязанный к вашему аккаунту Telegram.
  4. Следуйте инструкциям для завершения процесса аутентификации.
